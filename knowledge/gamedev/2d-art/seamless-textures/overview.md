# Seamless Textures - Overview

## What this covers

Seamless / tileable textures for 2D: offset-and-paint trick, mirror techniques, repetition reduction (variant tiles + decoration overlays), transitional tiles (corner / edge / T-junction), normal map authoring for 2D lighting (Sprite Lamp / Sprite DLight / Materialize), procedural base + hand-pixel overlay, roof/floor/wall specializations.

## When to use

This is a Phase A overview - quick orientation only. The full deep-dive
content (concept, code patterns, anti-patterns, common bugs) lives in the
dev-suite `SKILL.md` linked above. Topic-specific Phase B articles
