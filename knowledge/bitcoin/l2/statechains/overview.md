## What this covers

Statechains: off-chain UTXO ownership transfer via Statechain Operator (SO) signing. Mercury Layer (CommerceBlock) is the main implementation. 1-of-N security model.

dev-suite `SKILL.md` linked above. Topic-specific Phase B articles
