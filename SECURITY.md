## Reporting a Vulnerability

This repository contains only Markdown documentation files and no executable code. The attack surface is therefore very limited.

If you believe you have found a security-relevant issue (e.g., a link pointing to a malicious resource, sensitive data accidentally committed), please **do not** open a public GitHub issue.

Instead, report it privately by emailing the maintainers via the contact information listed on the [claude-dev-suite](https://github.com/claude-dev-suite) GitHub organisation page.

We will acknowledge receipt within **72 hours** and aim to resolve confirmed issues within **14 days**.
